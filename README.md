Overview:

The AI News Summarizer is a fully automated workflow built using n8n and Google Gemini AI.
It collects the latest AI and technology news from multiple sources, processes the information using AI, and delivers a structured intelligence briefing directly via email.
This project demonstrates real-world automation, API integration, and AI-powered content processing.

Objective:

To reduce information overload by:
Aggregating news from multiple trusted sources
Generating concise and structured summaries
Delivering a professional tech briefing automatically
The system runs without manual intervention.

Workflow Architecture:
Schedule Trigger:

Automatically runs every 3 days at 10 AM
Data Collection:
AI Business RSS Feed
TechCrunch RSS Feed
AI Events via SerpAPI (Google Events Engine)

Data Processing:

Merge multiple data streams
Aggregate structured content
Prepare input for AI summarization

AI Summarization:

Powered by Google Gemini API
Generates structured sections:
AI News Highlights
Technology Updates
Upcoming AI Events
Ensures professional and concise summaries

Email Delivery:

Automatically sends the formatted briefing via Gmail

 Technologies Used:

n8n (Workflow Automation)

API Integration & Data Aggregation

 Key Features:

 Fully automated execution
 Multi-source news aggregation
 AI-powered structured summarization

 Automated email intelligence report
 Scalable and modular design
